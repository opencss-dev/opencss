openCSS Features

Designed to be flexibile. openCSS offer a range of utility classes, mixins, and variables that can be customized and combined to create unique designs.

SCSS-based for easy extension, customization and maintenance, with optimized CSS compilation.

Grid system is built on a familiar 12 column fluid grid, so you avoid the need to learn something entirely new.

openCSS is an open-source project and 100% free for both private and commercial use.
